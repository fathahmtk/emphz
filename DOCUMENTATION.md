# EMPHZ - Website Blueprint Documentation

This document contains the strategic assets for the EMPHZ brand and website implementation.

---

## 1. DESIGN SYSTEM (SPECIFICATIONS)
- **Primary Theme:** Minimalist + Industrial + Futuristic.
- **Colors:**
  - Background (Dark): `#0A0A0A`
  - Accent (Electric Blue): `#00F2FF`
  - Border (Subtle): `rgba(255, 255, 255, 0.1)`
- **Typography:**
  - Headings: **Space Grotesk** (Weight: 500, 700)
  - Body: **Inter** (Weight: 400, 500)
- **UI Element Strategy:**
  - Use 1px borders to define structure (Industrial).
  - Use Backdrop Blurs (Glassmorphism) for modal and navigation depth.
  - Smooth 300ms transitions for all hover states.

## 4. CONVERSION (CTA) STRATEGY
- **Sticky Header:** "Get Quote" button always available in the top right.
- **Product-Level CTA:** "Spec Sheet" downloads lead to email capture.
- **Urgency:** Mention "Deploy in 48H" to emphasize speed over traditional construction.
- **Social Proof:** Testimonials placed immediately after the Product Categories preview.

---

## 5. MOBILE-FIRST BEHAVIOR
- **Stacking:** Grid layouts switch from 3-col to 1-col with 24px gap.
- **Touch Targets:** Buttons are minimum 48px height.
- **Navigation:** Full-screen overlay menu for ease of use on small screens.
- **Performance:** Images are lazy-loaded with a blur placeholder or dominant color.
