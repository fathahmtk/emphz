# App Setup

This repository includes everything you need to run the app locally.

## Run locally

**Prerequisite:** Node.js


1. Install dependencies:
   `npm install`
2. Add your Gemini API key to `.env.local`:
   `GEMINI_API_KEY=your_api_key_here`
3. Run the app:
   `npm run dev`
